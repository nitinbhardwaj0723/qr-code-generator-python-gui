# QR Code Generator

A desktop QR Code Generator built using **Python**, **Tkinter**, **qrcode**, and **Pillow**.  
This application allows users to generate, preview, and save QR codes from any text or URL through a simple and user-friendly GUI.

---

## 🚀 Features

- Generate QR codes from **text or URLs**
- Live **QR preview** inside the GUI
- Save QR code as **PNG image**
- Custom **file name**
- Clean and simple **desktop interface**
- Beginner-friendly Python project

---

## 🛠️ Technologies Used

- **Python**
- **Tkinter**
- **qrcode**
- **Pillow**
